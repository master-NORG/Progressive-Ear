Welcome! These are custom ear training tools I built for my own studies. After finding that the available apps were all falling short in one way or another, I decided to make my own with claude.ai. The tools are also based on suggestions from my piano teacher.

For now these tools are simple — limited in rhythmic and harmonic complexity. I'll update them or add more advanced versions as I progress.

Everything runs as a plain HTML page in any desktop or mobile browser, using ABC notation to display music (a much simpler format than MusicXML for straightforward use cases like these).
